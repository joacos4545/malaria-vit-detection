# Detección de Malaria usando Vision Transformer

Este repositorio corresponde al desarrollo de un proyecto académico de Ingeniería Informática
orientado a la detección automática de parásitos de malaria en imágenes microscópicas de
frotis sanguíneos, utilizando modelos basados en Vision Transformer.

## Estado del Proyecto
Actualmente, el proyecto se encuentra en la fase de análisis de imágenes, revisión teórica
y diseño conceptual del modelo. Se han definido la estructura del sistema, las técnicas de
preprocesamiento y los principales hiperparámetros a evaluar.

## Objetivos
- Analizar imágenes microscópicas de glóbulos rojos.
- Diseñar un modelo Vision Transformer para clasificación binaria.
- Evaluar el impacto de los hiperparámetros en el rendimiento del modelo.

## Tecnologías
- Python
- PyTorch (previsto)
- NumPy
- OpenCV
- Matplotlib

## Dataset
Se utiliza el conjunto de datos público "Malaria Cell Images Dataset (NIH)".
